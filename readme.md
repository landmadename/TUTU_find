#### 简介：

- **什么是TUTU find:**
  - 是基于微信小程序的， 高度可定制，可扩展的寻物系统。
- **为什么要用TUTU find：** 
  - ​	明天就要英语六级考试了，然而就是死活想不起来去年到底把考试耳机放在哪里了。	但是如果你用了TUTU find，你就可以直接问你的语音助手：“考试耳机在哪？”	她就会告诉你：“考试耳机 在 家里的大卧室的书架后面柜子的第三个抽屉里。”
  - ​         如果你是一个客似云来的商店的老板，在欣喜面对节节攀升的存款数额的同时，可能也会有一丝苦恼：无数的客人都抢着要问那可怜到忙不过来的店员，“xxxx有吗？”，“xxx在哪里！”         如果你在店里用了 TUTU find ，她就可以直接做一名合格的导购，告诉他们：“油画调色油在C架的第三层”
  - ​         如果你是一个仓库管理员，正好你也用了TUTU find，在下来新任务的时候，无需绞尽脑汁、大海捞针，只需要问一句：“4N30光耦在哪？”
  - ​         如果你在学校图书馆，鬼知道 TP311.56/864=2 放在哪里啊！
  - ​         如果你…………
- **为什么高度可定制，可扩展**
  - TUTU find 设计为 “终端	服务器	数据库”相互独立的模式，任何一个部件都可以简单的进行替换。
  - 比如说，仓库的管理软件不是普通的数据库，只要管理软件有接口，我们经过简单的封装，就可以像普通数据库一样被TUTU find调用
  - 比如说，如果不满足于在手机上操作，只需调用api就可以在家里的智能音箱上使用。若是更有需求，甚至可以放在商场的各个角落。
  - 服务器也可以任意更换，不会影响终端和数据库

#### 正式使用！

1. **域名&密钥**

   1. 正式使用的第一步一定是创造一个自己的独立空间

   2. 在第三页设置页中找到：

      > ![1560138687840](C:\CODE\python\TEMP\TUTU_find\imgs\1560138687840.jpg)

   3. 在上面的框中按要求填入使用的服务器的域名，也可以不修改，使用公用服务器。

   4. 修改密钥

      1. 同一个服务器中不同的数据需要不同的密钥来访问
      2. 密钥最长为128位
      3. 设置了新的密钥就会为你划分新的空间，密钥自然是越长越随机越安全
      4. 也可以使用微信登陆，让微信帮你生成随机密码
      5. 务必要保存好自己的密钥，这是唯一的凭证！ 

2. **录入数据**

   1. 接下来的工作就是录入数据

   2. 以下操作全都在编辑页面

   3. **新建**：

         > 容器里**只能存放东西或者嵌套容器**，在同一个容器里，容器和东西不能共存。可以参考测试数据理解。
         >
         > 容器的嵌套组成**路径**
